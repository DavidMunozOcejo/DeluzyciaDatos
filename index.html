07 / 09
Permanencia en Mesa
Análisis de tiempo de ocupación por mesa
⏳ Datos no disponibles en el sistema actual El sistema Square registra todos los artículos de cada ticket con el mismo timestamp (hora de cierre). No hay diferencia entre primer y último pedido, por lo que es imposible calcular tiempos de permanencia con los datos exportados.

Para activar este análisis: Square permite configurar el campo «Número de mesa» y registrar apertura y cierre de cada comanda. Con esos datos se calcularía automáticamente: tiempo medio de permanencia por turno, día de semana y restaurante.
